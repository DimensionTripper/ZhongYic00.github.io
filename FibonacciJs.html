Fibonacci

斐波那契数列的JavaScript实现方法

其实JavaScript有一个好处，就是通常不用担心数据类型问题，数据超大了会自动转换为科学计数法表示。


不多说了，直接上代码：



var n=prompt("Type in Data Range");
var i;
var fib=new Array();
fib[0]=0,fib[1]=1;
for(i=2;i<=n;i++)
{
fib[i]=fib[i-1]+fib[i-2];
document.writeln(i,' : ',fib[i]);
}
